# 69. Sqrt\(x\)

![](.gitbook/assets/image%20%2833%29.png)

## Analysis

Use binary search.

## My Code

```javascript
/**
 * @param {number} x
 * @return {number}
 */
var mySqrt = function(x) {
    if (x===1) return 1;
    let left = 0;
    let right = x;
    let middle = Math.floor((left+right)/2);
    while(left != middle){
        if (middle*middle===x){
            return middle
        }
        if (middle*middle > x){
            right = middle
        }else{
            left = middle
        }
        middle = Math.floor((left+right)/2);
    }
    return left
};
```
