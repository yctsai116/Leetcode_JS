## My Code

I use a hash table to solve this problem since the insertion and deletion of it are O\(1\).

```javascript
/**
 * @param {number[]} nums
 * @return {number}
 */
var singleNumber = function(nums) {
    let ans = {};
    for (let i = 0; i < nums.length; i++){
        if(ans[nums[i]]){
            delete ans[nums[i]]
        }else{
            ans[nums[i]]=true
        }
    }
    return Object.entries(ans)[0][0]
};
```

## Solution 1

Use bitwise operator to solve this problem.   
XOR operator is commutative; which means, A^B=B^A, also A^A=0.

### Bitwise operator

```javascript
// Bitwise AND (&)
const a = 5;        // 00000000000000000000000000000101
const b = 3;        // 00000000000000000000000000000011

console.log(a & b); // 00000000000000000000000000000001
// expected output: 1

// Bitwise OR (|)
const a = 5;        // 00000000000000000000000000000101
const b = 3;        // 00000000000000000000000000000011

console.log(a | b); // 00000000000000000000000000000111
// expected output: 7

//Bitwise XOR (^)
const a = 5;        // 00000000000000000000000000000101
const b = 3;        // 00000000000000000000000000000011

console.log(a ^ b); // 00000000000000000000000000000110
// expected output: 6
```

```javascript
var singleNumber = function(nums) {
    let c = 0;
    for (let ele of nums) {
        c ^= ele;
    }
    return c;
};
```
