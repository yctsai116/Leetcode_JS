## My Code

```javascript
/**
 * @param {string} s
 * @return {number}
 */
var titleToNumber = function(s) {
    let ans = 0;
    for(let i = 0; i < s.length; i++){
        ans+=(s.charCodeAt(i)-64)*26**(s.length-1-i)
    }
    return ans
    
};
```

transform alphabets into UTF-16 code unit and apply 26-base system.
